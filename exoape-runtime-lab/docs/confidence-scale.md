# Confidence Scale

## exact
Directly present in code/snippet with unambiguous semantics.

## observed
Visible behavior/pattern is clear, but complete ownership boundaries are partially implicit.

## inferred
Reasonable architectural conclusion from multiple observed signals, but no explicit definitive implementation line.

## unknown
No sufficient evidence yet.

---

## Required report format

For each important claim include:

1. confidence label
2. evidence snippet ID
3. source file path
4. short rationale
